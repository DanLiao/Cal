# SwiftCalculator

这是一个用 Swift 编写的简单计算器项目。

## 项目文件

*   `Cal.swift`: 包含计算器逻辑的 Swift 文件。
*   `Info.plist`: 项目的配置文件。
*   `Package.swift`: Swift 包管理器配置文件。
*   `calculator.icns`: 项目的图标文件。
*   `InfoPlist.strings`: 本地化字符串文件。
*   `LICENSE`: 项目的许可证文件。
*   `README.md`: 本文件，项目说明。
*   `test_float.txt`: 测试文件
*   `test_calculator.sh`: 测试脚本
*   `run_test.sh`: 运行测试脚本
*   `build_app.sh`: 构建app脚本
*   `record/`: 记录文件夹
*   `.gitattributes`: git属性文件
*   `.gitignore`: git忽略文件
*   `使用说明.md`: 使用说明文件

## 如何使用

1.  克隆此仓库到您的本地。
2.  使用 Xcode 打开项目。
3.  构建并运行项目。
